How to cope with no capture groups


Use look-aheads and look-behinds

.───────────────────────────────.
│    Syntax    │    Function    │
│──────────────│────────────────│
│              │                │
│ (?=PATTERN)  │ +ve Lookahead  │
│ (?!PATTERN)  │ -ve Lookahead  │
│ (?<=PATTERN) │ +ve Lookbehind │
│ (?<!PATTERN) │ -ve Lookbehind │
│              │                │
'───────────────────────────────'


Use \K to say, match but dont print everything before this


 * Poor man's ack
        > alias ack='grep -PbHnTr --binary-files=without-match  --color=always'

 * Count dpkg packages quicky
        > pkgs="$(grep -G '^Package:' -c /var/lib/dpkg/status)"
        > pkgs=$(( pkgs - 32 )) # to be consistent with apt list
