<h2>Useful flags</h2>

#TABLE Flag	Function 
-r	Recursive directory search
-o	Print only match
-P	Enable PCRE regex
-n	Line Number
-H	Filename
-b	Offset
-v	Invert (Give opposite)
#END TABLE

<h2>How to cope with no capture groups</h2>

Use look-aheads and look-behinds
#HTML-ESCAPE
#TABLE Syntax	Function
(?=PATTERN)	+ve Lookahead
(?!PATTERN)	-ve Lookahead
(?<=PATTERN)	+ve Lookbehind
(?<!PATTERN)	-ve Lookbehind
#END TABLE
#END HTML-ESCAPE

Use \K to say, match but dont print everything before this
